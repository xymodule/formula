# formula

支持对数学运算的编译执行，主要支持以下数学公式：

* 支持四则运算
* 支持三角函数
* 支持位运算
* 支持三元操作
* 支持逻辑运算符
* 支持自定义参数
* 支持自定义函数
* 支持公式缓存

支持lua
* 支持lua缓存
